# Compliance Audit Skill

**Trigger Condition**: Compliance review requested.

## Checks
- RBI controls
- DPDP controls
- SOC2 controls
- Data retention

## Execution Steps
1. Review the architecture or component against the designated compliance frameworks.
2. Verify RBI PPI guidelines are met.
3. Check for DPDP Act 2023 compliance (consent management, data residency).
4. Evaluate SOC2/ISO27001 readiness.
5. Validate data retention and audit log coverage.

## Output
- Gap analysis
- Control matrix
