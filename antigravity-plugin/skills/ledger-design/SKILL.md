# Ledger Design Skill

**Trigger Condition**: Financial workflow design needed.

## Input
- Financial workflow

## Execution Steps
1. Analyze the financial workflow for money movement.
2. Design double-entry bookkeeping schema.
3. Structure the journal entries for the workflow.
4. Design the balance reconciliation flow.

## Validation
- Ensure Debit == Credit for every transaction.
- Ensure all ledger entries are immutable.

## Output
- Double-entry schema
- Journal structure
- Reconciliation flow
